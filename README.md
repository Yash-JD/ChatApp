# 💬 Realtime Responsive Chat App  
A feature-rich, responsive, and real-time chat application built with modern web technologies.

---

## 🚀 Features

### **Frontend**
- **React.js**: Built for an interactive and dynamic user experience.  
- **ShadCN**: Modular UI components for a professional design.  
- **Tailwind CSS**: Beautiful and responsive styling.  
- **Fully Responsive Design**: Works seamlessly across devices (Mobile 📱, Tablet, Desktop 💻).  

### **Backend**
- **Node.js and Express.js**: Handles real-time communication and RESTful APIs.  
- **Socket.io**: Powers real-time messaging for a seamless chat experience.  
- **Multer**: Handles file and image uploads securely.  

### **Database**
- **MongoDB**: Efficient and scalable storage for messages, files, and user data.  

### **Core Features**
- **Real-time Messaging**: Instant chat updates powered by WebSockets.  
- **Group Chats and Personal Messages**: Chat with friends or in groups.  
- **File Uploads and Downloads**: Share files and media easily.  
- **Emoji Support**: Fun and expressive conversations.  
- **Authentication**: Secure login with JWT Tokens.  
- **State Management**: Handled seamlessly using Zustand.  

---

## 🛠️ Tech Stack

- **Frontend**: React ⚛️

## 🌐 API Endpoints  

### **Authentication**  
- `POST /api/auth/register`: Register a new user.  
- `POST /api/auth/login`: Login user and generate JWT.  

### **Chat**  
- `GET /api/chat/:id`: Fetch all messages for a chat.  
- `POST /api/chat`: Send a new message.  

### **File Upload**  
- `POST /api/upload`: Upload a file.
